# Marketing Calendar UI Specification

## Primary views

- Campaign Calendar
- Promotion Calendar
- Content / Creative Calendar
- New Item Launch Calendar
- Branch Opening Calendar
- Mall Event Calendar
- Holiday / Seasonal Calendar

## Required calendar controls

- Date range navigation
- Month/week/agenda presentations as appropriate
- Filters: brand, branch, campaign, promotion type, assigned user, status, channel, date range
- Clear all-day/date-only versus scheduled-time visual treatment
- Readable status and readiness indicators
- Click-through to authoritative campaign/promotion/launch/creative record

## Event information

Calendar cards should show the minimum decision context:

- Title
- Type
- Brand and branch scope summary
- Owner
- Status
- Date/range
- Required approval/readiness indicator where relevant

## Mobile priorities

- Today/upcoming/overdue work
- Campaign/promotion date and assigned tasks
- Quick status/comment/evidence update
- Filter by user/brand/branch
- Avoid dense month-grid interactions as the only mobile experience.
