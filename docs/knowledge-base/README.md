# OGFI ERP Knowledge Base

This directory contains role-based, user-facing guidance for using the OGFI ERP. It is derived from approved specifications and verified implemented behavior.

## Start here

- `getting-started/` — signing in, selecting a working location, dashboard basics, notifications and statuses
- `purchasing/` — Purchase Requests, approvals, Purchase Orders, supplier-facing tasks
- `warehouse-inventory/` — receiving, transfers, stock counts, wastage, inventory controls
- `projects/` — Project Tracker tasks, blockers, milestones, risks, reports, and linked ERP records
- `projects/managing-expansion-projects-and-opening-readiness.md` — Expansion Dashboard, Site Pipeline, lifecycle gates, permits/documents, construction, opening readiness, punch list, report rollups, and source-record boundaries
- `finance/` — controlled finance and workforce foundations such as cash advances, bank/cash readiness, workforce readiness, evidence references, and non-posting settlement boundaries
- `approvals/` — review, approval, rejection, and return-for-revision guidance
- `branch-operations/` — branch-facing operational tasks as later phases are released
- `administration/` — user, role, scope, release-readiness, privileged MFA, session invalidation, break-glass, and master-data guidance for authorized administrators
- `reports/` — filters, exports, saved views, and report interpretation
- `troubleshooting/` — known issues and safe recovery steps
- `glossary/` — approved ERP terms
