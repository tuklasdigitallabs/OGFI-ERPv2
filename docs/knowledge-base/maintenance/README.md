# Maintenance

## Review ticket activity

Open a maintenance ticket to review its read-only Activity section. Activity is immutable audit history for the ticket and cannot be edited or deleted from the workspace.

- The maintenance ticket register is paginated on the server. Search, requested date, status, and priority filters remain selected while moving between pages; dashboard summary cards and CSV export are separate views.
- Maintenance dashboard and task results always remain limited to the selected company and location. When no brand is selected, the workspace uses the company/location scope rather than treating an empty brand value as a real brand identifier.
